# StoreQueue
## Technologies Used
- React
- Node.js
- Express
- PostgreSQL
- Tailwind CSS

## Getting Started
```
- Clone the repository
- cp .env.example .env
- npm install
- npm run dev
```

## Pertimbangan
- Pemilihan tech stack, karena lebih terbiasa dan lebih menguasai tech stack tersebut.
- Menggunakan arsitektur projek FE & BE dipisah, karena lebih modern dan lebih fleksibel, bisa untuk projek kecil dan projek besar.
- Menggunakan Tailwind CSS, karena untuk mempermudah dan mempercepat proses design UI sederhana.
- Menggunakan .env.example untuk memberikan gambaran besar terkait dengan isi dari .env (isi value bisa dilihat di pdf yang sudah dikirimkan)
